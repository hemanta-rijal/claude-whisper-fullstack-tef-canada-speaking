# Full stack (scaffold)

Monorepo layout for learning: **Node.js** in `backend/`, **Angular** in `frontend/` (you generate the app when ready).

## Backend

```bash
cd backend
npm run dev
```

Runs **Express** (`src/index.js`). Visit `http://localhost:3000` — you should see JSON.

## Frontend

See `frontend/README.md`. The folder is intentionally empty except for that note until you run the Angular CLI.
